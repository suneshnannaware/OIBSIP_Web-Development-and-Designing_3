## Objective
The objective of this task was to develop a Temperature Converter web application using HTML, CSS, and JavaScript.  
The goal was to build an interactive tool that converts temperature values between Celsius, Fahrenheit, and Kelvin.

---

## Steps Performed
- Created the basic structure using HTML.
- Designed a user-friendly interface with input field, dropdown menu, and convert button.
- Implemented temperature conversion logic using JavaScript.
- Added input validation to handle empty values.
- Styled the application using CSS with centered layout and background design.

---

## Tools & Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- VS Code  
- Google Chrome  

---

## Features
- Convert between Celsius, Fahrenheit, and Kelvin  
- Simple and clean UI  
- Input validation  
- Instant result display  
- Responsive centered layout
